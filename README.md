# Java Programming I — Part 1

My solutions for **Part 1** of the [Java Programming I](https://java-programming.mooc.fi/part-1) MOOC by the University of Helsinki ([mooc.fi](https://mooc.fi/)).

## About the course

Java Programming I is a free, open online course that introduces programming fundamentals using Java. Part 1 covers the very first concepts: printing, variables, conditional statements, basic input/output, and the core building blocks of any program.

## Topics covered in Part 1

- Printing output with `System.out.println`
- Reading user input with `Scanner`
- Variables and data types (`int`, `String`, `double`, `boolean`)
- Arithmetic operations
- Conditional statements (`if`, `else if`, `else`)
- Logical operators (`&&`, `||`, `!`)

## Repository structure

Each folder corresponds to one exercise from the course, named according to the official MOOC convention (`part01-Part01_XX.ExerciseName`).

## Tech

- **Language:** Java
- **Tooling:** TMC (Test My Code) plugin, VS Code
